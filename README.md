# SFD_BiznesElektroniczny
## To run scraper, use:
`cargo run`

# Uruchomienie projektu
Aby uruchomic projekt musisz znajdowac sie w folderze shop-src
## Uruchomienie kontenera
```bash
make up
```

## Wczytanie bazy danych
```bash
make restore
```

## Wylaczenie kontenera
```bash
make down
```

## Zrobienie zrzutu bazy
```bash
make dump
```
